<h1>Tenkai</h1>
Made for learning and personal use purposes. This app gives the user the ability to manage their <a href="https://anilist.co">AniList</a> anime list.
<br>
<br>
<small><i>Warning: still a prototype, not fully developed for general use.</i></small>
<h3>Features</h3>
<ul>
<li>Offline caching</li>
<li>Browse, add, edit, search for and manage shows</li>
<li>Find users and see stats and information regarding their profile</li>
<li>Add your official desired sources to watch your shows</li>
<li>Airing notifications</li>
</ul>
<h3>Project details</h3>
<ul>
<li>Offline caching: <b>Room</b></li>
<li>Remote data source: AniList GraphQL Database with <b>Apollo</b></li>
<li>Modularization: <b>Layered feature</b></li>
<li>Structure: <b>Clean Architecture with MVVM</b></li>
<li>User Interface: <b>Jetpack Compose</b></li>
<li>Data presentation: Paginated with <b>Paging 3 library</b></li>
<li>Preferences: <b>DataStore</b></li>
</ul>
